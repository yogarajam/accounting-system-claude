Instructions

Enter bank statement transactions manually:

  • Statement Date: Date shown on the bank statement
  • Transaction Date: Actual date the transaction occurred
  • Description: Transaction description from the statement
  • Reference: Check number or transfer reference
  • Debit: Money leaving the account (withdrawals, payments)
  • Credit: Money entering the account (deposits, transfers in)
Enter either a debit OR credit amount, not both.
